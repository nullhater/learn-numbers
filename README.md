# Learn Numbers

Небольшое браузерное приложение для тренировки записи чисел словами на английском и немецком.

## Что умеет

- выбор языка ответа: English или Deutsch
- настройка собственного диапазона от `0` до `9999999`
- быстрые пресеты диапазона
- режим на заданное количество заданий
- мгновенная проверка ответа
- показ правильного варианта при ошибке
- итоговая статистика по правильным и неправильным ответам

## Запуск

Можно просто открыть файл `index.html` в браузере.

Если удобнее через локальный сервер:

```bash
python3 -m http.server 8000
```

После этого открой `http://localhost:8000`.
