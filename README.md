# Learn Numbers

A small browser-based app for practicing how to write numbers in words in English and German.

## Features

- answer language selection: English or German
- interface language selection: English, Russian, or German
- custom number range from `0` to `9999999`
- quick range presets
- training mode with a configurable number of tasks
- instant answer checking
- immediate display of the correct answer after a mistake
- final statistics with correct and incorrect answers

## Run Locally

You can open `index.html` directly in your browser.

If you prefer running it through a local server:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Deploy to GitHub Pages

This project is a static single-page application, so it can be deployed easily with GitHub Pages.

1. Push the repository to GitHub.
2. Open the repository settings.
3. Go to `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your branch, usually `main`, and the root folder `/`.
6. Save the settings and wait for the site URL to appear.
